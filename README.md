# BreakoutCS50
Pset3 CS50 2015x

Harvard CS50 pset3 : "Breakout" solution for this Pset. Breakout is our first program running graphical user interface (GUI) in CS50.
A little game with a ball , a paddle and some bricks. 
The goal is simple : just bouce the ball off of the paddle so as to hit the bricks with it.

-------------------------------

Breakout est un petit et simple jeu ou l'on doit frapper des briques avec une petite balle en mouvement qui rebondit sur une plateforme que nous controlons.
Il s'agit du premier programme utilisant (GUI) dans le CS50x 2015, écrit en C.
